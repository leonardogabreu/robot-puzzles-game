# 🤖 Robot Puzzles Game

An educational "Serious Game" focused on teaching children programming logic and developing their motor coordination.

## 🎓 About the Project
This prototype is being developed as part of an Undergraduate Research Project in the Computer Science program at the Federal University of Uberlândia (UFU), under the supervision of Professor Rita Maria da Silva Julia. The main objective is to use game mechanics to create an interactive and engaging learning environment for children.

## ⚙️ Implemented Mechanics (MVP)
The game is currently in its first stable version (Minimum Viable Product) and features:
- **Grid Movement:** Precise navigation system within the scenario.
- **Score and Combo System:** Dynamic rewards based on consecutive correct deliveries.
- **Visual Feedback (Floating Text):** 3D floating texts indicating score changes in real-time.
- **Match Management:** A dynamic 60-second timer and complete control over game states (Start, Playing, Game Over).

## 💻 Technologies Used
- **Engine:** Unity 6
- **Language:** C#

## 🚀 How to Run the Game
Since this is the project's source code, you will need the game engine to test it on your machine.

1. Ensure you have **Unity Hub** and the corresponding **Unity** version installed.
2. Clone this repository using your terminal:
   ```bash
   git clone [https://github.com/leonardogabreu/robot-puzzles-game.git](https://github.com/leonardogabreu/robot-puzzles-game.git)

3. Open Unity Hub, click on Add (Add project), and select the cloned folder.
4. With the project loaded, navigate to the Assets/Scenes folder in the Project tab.
5. Double-click the main scene, and then press the Play button on the top bar of the editor.
